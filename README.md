# Resumos Estruturados

***Como o Python surgiu***

O Python foi criado por Guido van Rossum no final da década de 1980, com o objetivo de desenvolver uma linguagem mais simples, intuitiva e eficiente do que as existentes na época. Seu desenvolvimento começou em 1989 como um projeto pessoal, inspirado em outras linguagens como ABC, C e Perl. A primeira versão oficial foi lançada em 1991. O nome da linguagem foi inspirado no grupo de comédia britânico Monty Python, refletindo a proposta de ser uma linguagem leve e acessível.

*Desde sua criação, o Python prioriza:*

- Legibilidade do código
- Sintaxe simples
- Uso de indentação para organização

***Passos para começar em Python***

- Preparação do ambiente
- Instalar o Python (python.org)
- Configurar o PATH no Windows
- Instalar o editor de código (VS Code)

***Primeiros passos***

- Criar arquivos .py
- Usar comandos básicos:
- print() → exibir informações
- input() → receber dados

***Fundamentos da linguagem***

- Variáveis e tipos de dados:
- int, float, string, bool

***Estruturas de controle***

Condicionais:
- if, elif, else

Repetições:
- for, while

***Estruturas de dados***

- Listas → coleção de valores

- Dicionários → chave e valor

***Evolução prática***

Criar pequenos projetos:

- Calculadora
- Cadastro simples
- Validador de senha

***Uso de bibliotecas***

- Aproveitar códigos prontos da comunidade:
- Pandas → dados
- Streamlit → aplicações web

***Frameworks Web em Python***

O Python possui diversas ferramentas para desenvolvimento web:

**Django:**

- Framework completo e robusto
- Ideal para sistemas grandes
- Possui recursos prontos (ORM, autenticação, etc.)

***Flask***

- Micro-framework leve
- Mais flexível
- Indicado para projetos menores ou personalizados

***FastAPI***

- Focado em APIs
- Alto desempenho
- Muito utilizado em sistemas modernos

***Streamlit***

- Biblioteca para criar interfaces web
- Ideal para iniciantes
- Não exige conhecimento em front-end

# Glossário de Conceitos

- Linguagem de Programação - Forma de dar instruções ao computador
- Variável - Espaço que armazena dados
- Tipo de Dado - Define o tipo da informação (texto, número, etc.)
- Função - Bloco de código reutilizável
- Condicional - Estrutura que toma decisões (if/else)
- Loop - Estrutura de repetição
- Lista - Coleção de elementos
- Dicionário - Estrutura de dados com chave e valor
- Biblioteca - Conjunto de códigos prontos
- Framework - Estrutura que facilita o desenvolvimento
- API - Interface de comunicação entre sistemas
- IDE - Ambiente de desenvolvimento (ex: VS Code)
